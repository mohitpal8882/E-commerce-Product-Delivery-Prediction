# E-commerce-Product-Delivery-Prediction
•	Built an end-to-end ML pipeline to predict on-time delivery for an international e-commerce company using a dataset of 10,999 shipment records.
•	Performed comprehensive EDA identifying key delay factors: shipping mode, product weight, customer priority, and prior customer calls — insights that informed feature selection.
•	Implemented and compared 4 classification models (Logistic Regression, Decision Tree, KNN, Random Forest); achieved highest accuracy of ~69% with Random Forest after hyperparameter tuning.
•	Analyzed feature importance scores to deliver actionable logistics recommendations, highlighting that warehouse block and product weight were top predictors of delay.
•	Created full model evaluation reports including confusion matrix, ROC-AUC, and precision-recall analysis using Matplotlib and Seaborn.
